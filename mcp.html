AIと外部ツールをつなぐ
MCP

Anthropicが提唱するオープンプロトコル。AI Agentが外部のツールやデータソースとシームレスに連携する標準規格です。

MCPとは？

Model Context Protocol（MCP）は、Anthropicが2024年11月に公開したオープンプロトコルです。LLMアプリケーションが外部のデータソースやツールと接続するための標準化された仕組みを提供します。

従来、AI Agentがツールを利用するには個別のAPI統合が必要でしたが、MCPにより「つなぎ方」が統一されました。USBポートのように、一つのプロトコルで様々なツールを接続できます。

アーキテクチャ

MCPはHost / Client / Serverの3層構造で動作します。HostがClientを管理し、ClientがServerと1対1で通信します。

🖥️
Host
LLMアプリケーション本体。Claude DesktopやIDEなど
🔌
Client
Host内部で動作し、Serverとの接続を管理する
⚙️
Server
外部ツールやデータソースの機能を公開する

主要な機能

MCPサーバーは3種類のプリミティブを通じて機能を公開します。

🔧

Tools

LLMが呼び出せる関数。ファイル操作、API呼び出し、データベースクエリなど、外部システムへのアクションを実行します。

📂

Resources

LLMが参照できるデータソース。ファイル内容、データベースのレコード、APIレスポンスなど、コンテキストとして読み込めます。

💬

Prompts

再利用可能なプロンプトテンプレート。特定のタスクに最適化されたプロンプトをサーバー側で定義し、クライアントから呼び出せます。

自作MCPサーバー

MCPサーバーはPythonやTypeScriptの公式SDKを使って簡単に構築できます。以下はPython（FastMCP）での最小構成の例です。

from mcp.server.fastmcp import FastMCP

# サーバーを作成
server = FastMCP("my-server")

# ツールを定義
@server.tool()
def greet(name: str) -> str:
    """指定された名前に挨拶を返す"""
    return f"こんにちは、{name}さん！"
  • Python SDK — pip install mcp でインストール。FastMCPによるデコレータベースの簡潔な記述
  • TypeScript SDK — npm install @modelcontextprotocol/sdk でインストール。Node.js環境で動作
  • トランスポート — stdio（ローカル）とSSE/Streamable HTTP（リモート）の2種類をサポート
